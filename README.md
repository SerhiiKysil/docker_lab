# Аналіз зареєстрованих авто в Україні (Docker Microservices)

Цей проєкт є лабораторною роботою, що демонструє побудову мікросервісної архітектури за допомогою Docker Compose. Система автоматично завантажує набір даних у базу PostgreSQL, проводить аналіз якості, досліджує дані, будує графіки та виводить результати на веб-дашборд.

## 🏗 Архітектура проєкту

Проєкт складається з 5 контейнерів:
1. **db (PostgreSQL)**: База даних для зберігання сирих даних про автомобілі.
2. **data_load**: Зчитує `dataset.csv` та завантажує дані в БД.
3. посилання на датасет: https://data.gov.ua/dataset/06779371-308f-42d7-895e-5a39833375f0/resource/3f13166f-090b-499e-8e23-e9851c5a5f67
3. **data_quality_analysis**: Перевіряє дані на пропуски, дублікати та типи колонок (зберігає JSON-звіт).
4. **data_research**: Проводить аналітику (топ марок, типи пального, середній рік) і генерує JSON-звіт.
5. **visualization**: Будує графіки (PNG) на основі даних з БД за допомогою `matplotlib` та `seaborn`.
6. **web**: Flask-сервер, який об'єднує всі звіти, графіки та зразок даних у єдиний веб-дашборд.

## 🚀 Як запустити

1. Переконайтеся, що у вас встановлено Docker та Docker Compose.
2. Склонуйте репозиторій або перейдіть у папку з проєктом.
3. У кореневій директорії виконайте команду:
   ```bash
   docker compose up --build
